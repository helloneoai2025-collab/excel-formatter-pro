# Excel Formatter Pro

ระบบประมวลผล Master Form พร้อมข้อมูล Color แบบอัตโนมัติ

## 🎯 ฟีเจอร์
- ✅ ดึง PO# อัตโนมัติ
- ✅ เติมวันที่ปัจจุบัน
- ✅ แยก Color Code
- ✅ คงฟอร์ม 100%
- ✅ รองรับหลายไฟล์พร้อม ๆ กัน

## 🚀 วิธีใช้งาน
1. อัพโหลด Master Form
2. อัพโหลดไฟล์ข้อมูล (1 หรือหลายไฟล์)
3. กด "ประมวลผล"
4. ดาวน์โหลดไฟล์ output

## 📋 ไฟล์ที่จำเป็น
- streamlit_app.py - Main Application
- requirements.txt - Python Dependencies
- .streamlit/config.toml - Streamlit Configuration

## 💻 รันแอพ (Local)
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## 🌐 Deploy บน Streamlit Cloud
1. Push ไฟล์ไปยัง GitHub
2. เชื่อมต่อ Streamlit Cloud กับ GitHub
3. Deploy!
